# intern1_project2
Created a MySQL database named college_admission with tables: Students, Courses, and Applications.
Established Java–MySQL connectivity using JDBC.
Developed a student registration module to store student details (name, email, marks).
Added functionality to insert and manage courses with cut-off marks.
Implemented a course application system for students.
Designed merit evaluation logic to compare student marks with course cut-offs.
Built an admin panel (console-based) to view and manage applications.
Enabled approval/rejection of applications based on merit criteria.
Generated admission list output in CSV format.
Tested the system with sample data to ensure proper functionality.
